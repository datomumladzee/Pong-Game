

# 🏓 Python Pong Game

A classic Pong game built with Python’s **Turtle graphics** module.
The project uses Object-Oriented Programming (OOP) and supports both **Single Player (AI)** and **Two Player** modes.

---

## 🚀 Features

* 🎮 Single Player (vs AI)
* 👥 Two Player local multiplayer
* ⚡ Dynamic ball speed (increases after paddle hits)
* 🧮 Live scoreboard
* ➖ Center dashed line
* 🎯 Collision detection (walls & paddles)
* 🧠 Clean modular structure

---

## 🛠 Requirements

* Python 3.x
* No external libraries needed

---

## ▶️ How to Run

```bash
git clone https://github.com/your-username/pong-python.git
cd pong-python
python main.py
```

---

## 🎮 Controls

| Action    | Left Paddle | Right Paddle |
| --------- | ----------- | ------------ |
| Move Up   | W           | ↑ Arrow      |
| Move Down | S           | ↓ Arrow      |

### Menu Selection

* Press **1** → Single Player (AI)
* Press **2** → Two Players
